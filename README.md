#picoCTF Writeups

I am currently studying Cyber Security and using Capture The Flag (CTF) challenges to develop hands-on technical skills in vulnerability analysis and exploitation.

This repository documents my solutions and technical writeups for challenges from picoCTF. The goal of this repository is to demonstrate my approach to solving CTF challenges and to track my progress in developing practical cybersecurity skills.

Each writeup includes:
- An overview of the challenge
- Initial observations and reconnaissance
- Vulnerability analysis
- Exploitation steps
- Supporting scripts or tools where applicable
- Lessons learnt

Writeups are organised by challenge category and difficulty level to make navigation easier.

Through these challenges I have practiced techniques related to:
- Web exploitation
- Cryptography
- Reverse engineering
- Binary exploitation
- Network analysis
- Linux and Python scripting
